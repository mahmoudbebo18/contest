Welcome to Monitor+

Monitor+ is a Web API that allows easy access to a huge range of market data from cloud data servers that store historical reference data from stock exchanges, commodity exchanges, proprietary market data providers, independent pricing agencies as well as currencies, world indices and more. The Monitor+ API site between the subscriber and the cloud data servers enabling users to request and receive time series market data in standard JSON format.

With simple flexible licensing, Monitor+ allows easy integration of market data into new or existing platforms, information systems and trading systems. It is easy to use, extremely powerful and has native access in both Python and "R" to enable data scientists, analysts and developers to use it straight out of the box.

How it works:

A clients subscribes to one or more datasources and receives an API key that gives access to the service. The client sends a HTTP request for data and the server returns the market data in JSON format. It's that simple. With just two HTTP requests you can authenticate with the server and get values for any number of datasets.

The API is well documented with sample code and provides easy access to the metadata (data that describes the datasources and datasets) and values. Custom averages and calculated columns are both supported.

Why Monitor+?

Monitor+ is a RESTful API that is designed to enable fast and easy access to a vast amount of global reference market data plus the analytic tools to enable fast integration into both new and existing business information and trading systems. Monitor+ makes it easy to build reports, quote pages and create both in-house and commercial analytic solutions.

As well as offering low cost subscriptions to significantly reduce data costs, we provide flexible and simple licensing to allow internal use of the market data to maximize the customers return on the data.

We provide subscribers with direct access to "R", Python (in Jupyter notebooks) which enables access to a huge range of readily available analytic functions and reports. Using Monitor+, even the smallest companies can now have the analytic power that was unheard of only a few years ago. Averages, correlations, regressions, seasonality, forecasting, machine learning, MACD, ANOVA, ARIMA, SARIMA and thousands more analytic functions are freely available. We provide code samples to demonstrate just how easily powerful reports can be created. With Monitor+, market data is made easy.

We provide customers with the powerful “XLerate” for Excel add-in to enable easy access to market data for traders, analysts and front/mid or back office use. XLerate brings the power of Monitor+ to Excel and makes it easy to create both analytic reports and technical analysis charts using a 'point and click' interface. Reports can be shared and refreshed at the click of a button. If you can use Excel, you can use XLerate. Please look to the XLerate page under the "Solutions" menu at the top of the page for more information.

Additional API Features

JSON Result Format

The Monitor+ API is ideal for creating multiple web applications and reports as market data can be returned in one of three formats

  • Single Date. Ideal for creating "Quote page" style reports that require values for any one date including optional "Spark Line" and "change" values.
  • Historical TS. Market data is returned as individual time series.
  • Historical RC. Market data is returned in a ”row x column” format to make report creation easy.

Integration

The market data is directly accessible in "R" and Python or using our XLerate for Excel add-in. Use XLerate to create, update and share detailed Excel market reports. See the “Products” menu above for detailed information on XLerate for Excel. You can get more information on the “R” , Python packages and the API from the “Documentation” menu at the top of the page.

Averages

The API can return values as daily, weekly, half monthly, monthly, quarterly, half yearly and yearly. Custom averages can be calculated for weeks, half months and months. Custom quarterly, half-yearly and yearly averages can be created using custom month averages. As well as choosing date ranges, you can exclude (or move) weekend values, adjust weekend dates in average results and return the first or last date for any average. Fill options permit the filling of missing values with the previous, mid point or interpolated values. If the data vendor supports the feature, details of any published corrected values are also returned in the result.
